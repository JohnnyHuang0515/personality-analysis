## 🚀 部署與運行

### 開發環境啟動
```bash
# 後端啟動
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 前端啟動
cd frontend
npm start
```

### Windows 一鍵啟動
```batch
# 執行 start.bat
start.bat
```

### 測試驗證
```bash
cd backend
python test_comprehensive_analysis.py
python test_new_api.py
```
