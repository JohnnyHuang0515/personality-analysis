## 1. 專案概述 (Project Overview)

### 1.1 一行說明 (One-Liner)
*   一個整合多種人格理論的線上平台，旨在幫助使用者深入探索自我、解鎖潛能，並提供實用的個人與團隊發展建議。

### 1.2 問題陳述 (Problem Statement)
*   在快速變化的現代社會中，個人和團隊往往面臨自我認知不足的挑戰。這可能導致職業發展迷茫、人際溝通障礙，以及團隊協作效率低下。市面上雖然有多種人格特質測驗工具，但它們通常獨立存在，缺乏整合性，使得使用者難以全面且深入地理解自身的多元特質。此外，這些工具的解讀多為靜態報告，缺乏互動性和實用性的建議，無法有效引導使用者將人格洞察轉化為實際行動。

### 1.3 建議解決方案 (Proposed Solution)
*   我們將開發一個綜合人格特質分析平台，整合 MBTI、DISC、五大人格和九型人格等多種主流人格評估工具。本平台不僅提供一站式的測驗服務，更重要的是，它將透過智能演算法對不同測驗結果進行交叉分析和比對，生成綜合性的人格報告。這份報告將以視覺化圖表呈現，並提供個人化的解讀與實用建議，涵蓋職業選擇、溝通策略、團隊協作及個人成長等面向。此外，平台將提供互動式工具，例如情境模擬、發展計畫追蹤等，協助用戶將人格洞質轉化為具體的實踐。

### 1.4 目標用戶 (Target Users / Audience)
*   **個人使用者:** 尋求自我探索、職業發展規劃、提升人際溝通能力的大學生、職場新鮮人及各行各業的專業人士。
*   **企業/組織:** 希望優化團隊建設、提升員工績效、進行人才招聘與發展的人力資源部門及團隊領導者。
*   **教育機構:** 應用於學生輔導、生涯規劃及課程設計。

### 1.5 專案目標與成功指標 (Goals & Success Metrics)
*   **主要目標 (Primary Goals):**
    *   **提供一站式整合服務:** 整合至少四種主流人格測驗（MBTI、DISC、五大、九型），提供使用者便捷的綜合評估入口。
    *   **生成多維度綜合報告:** 開發智能演算法，對不同測驗結果進行深度交叉分析，產出獨特的綜合人格洞察報告。
    *   **提供實用行動建議:** 針對報告結果，提供個人化、可操作的職業、溝通與成長建議。
    *   **增強使用者互動性:** 引入互動式功能，如情境演練、發展計畫追蹤等，鼓勵使用者將洞察付諸實踐。
*   **成功指標/KPIs (Key Performance Indicators):**
    *   **使用者滿意度:** 使用者對綜合報告的滿意度達到 85% 以上（透過問卷調查）。
    *   **報告準確性:** 經專家評估，綜合報告的解讀準確性與實用性獲得高分。
    *   **使用者參與度:** 每月活躍使用者數量穩定成長 15%。
    *   **功能使用率:** 互動功能（如情境模擬、計畫追蹤）的使用率達到 60% 以上。
*   **(選填) 非目標 (Non-Goals):**
    *   本階段不包含線下諮詢服務的配對。
    *   本階段不開發原生手機應用程式 (APP)，專注於 Web 網站。

## 目前進度與現況總結（2024-07-17）

### 1. 系統架構與技術棧
- **後端**：FastAPI + SQLite，採用 SQLAlchemy ORM，API 路徑統一加上 `/api/v1` 前綴。
- **前端**：React + TypeScript + Tailwind CSS，已完成主要頁面與 API 串接。
- **資料管理**：題庫與分析規則皆有專屬 Python 檔案與資料庫表，支援多種人格測驗。

### 2. 題庫狀態
- **MBTI**：80 題，全部為兩個選項（A/B），符合 MBTI 二元對立設計。
- **DISC**：80 題，四個選項（A/B/C/D）。
- **Big5**：80 題，四個選項（A/B/C/D）。
- **Enneagram**：60 題，四個選項（A/B/C/D），已補齊最後一題。
